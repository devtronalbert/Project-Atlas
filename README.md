# Project Atlas

**Project Atlas** is a lightweight custom WordPress portfolio theme and installer plugin built for my personal website, [devonalbert.com](https://devonalbert.com).

The goal of this project was to create a fast, clean, professional portfolio site without relying on paid WordPress themes, page builders, or plugin-heavy layouts.

## Purpose

Project Atlas is designed to showcase professional experience across:

- Enterprise infrastructure
- Software development
- Systems integration
- Cybersecurity
- Networking
- Databases
- E-commerce platforms
- Business process automation

## Installation

### Theme

1. Zip the `theme/project-atlas` folder.
2. In WordPress, go to **Appearance → Themes → Add New → Upload Theme**.
3. Upload the ZIP.
4. Activate **Project Atlas**.

### Installer Plugin

1. Zip the `installer-plugin/project-atlas-installer` folder.
2. In WordPress, go to **Plugins → Add New → Upload Plugin**.
3. Upload the ZIP.
4. Activate the plugin.
5. Go to **Tools → Project Atlas Installer**.
6. Run the installer once.
7. Deactivate and delete the installer plugin after use.
