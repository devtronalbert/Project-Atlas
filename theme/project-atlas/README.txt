Project Atlas WordPress Theme v1.0.0

Install:
1. WordPress > Appearance > Themes > Add New > Upload Theme.
2. Upload project-atlas-theme.zip.
3. Activate Project Atlas.
4. Then install/run the Project Atlas Installer plugin.
5. Upload your final resume PDF to the web root as /resume.pdf or update the resume URL in functions.php.

Notes:
- Navigation links are real pages, not one-page anchor links.
- Projects are managed from the WordPress admin under Projects.
- No paid theme or page builder required.
