Project Atlas Installer v1.0.0

Install:
1. Activate the Project Atlas theme first.
2. WordPress > Plugins > Add New > Upload Plugin.
3. Upload project-atlas-installer.zip.
4. Activate it.
5. Go to Tools > Project Atlas Installer.
6. Click Install / Update Project Atlas.
7. Review pages and projects.
8. Deactivate and delete this installer plugin after use.

This installer uses WordPress functions, not raw SQL.
